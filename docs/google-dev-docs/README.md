# Google Search Central docs — local knowledgebase

Local, grounded copies of the Google developer documentation that is relevant to
robots.txt and Google's crawlers, so agents/developers can read authoritative context
without re-fetching. Generated by `tools/google-dev/pull-docs.sh`:

- `<slug>.md` — extracted text with a source/license attribution header. **Committed.**
- `rawhtml/<slug>.html` — the raw fetched page. **Gitignored**; rerun the script to
  rematerialize.

HTML→text conversion.

The `.md` files are generated artifacts — if an extraction problem turns up, fix
`tools/google-dev/pull-docs.sh` and re-run (or note the problem here) rather than
hand-editing them.

## Re-running / extending

```sh
# refresh the seed set (the 8 pages above):
tools/google-dev/pull-docs.sh

# pull additional pages (each becomes <slug>.md + rawhtml/<slug>.html):
tools/google-dev/pull-docs.sh https://developers.google.com/search/docs/crawling-indexing/<page>
```

To add pages permanently, extend the `PAGES` array in `tools/google-dev/pull-docs.sh`
and update the index table above.
